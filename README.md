# Cloud of dream

Simple project - MP3 player with minimal functionality.

App imports all audio files from the internal memory and SD-card. Player functionality: 
* Play song
* Pause
* Shuffle
* Repeat
* Progress Bar manipulations
* Backward
* Forward
* Next
* Previous
